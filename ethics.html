Guidelines for Developers

Developers are expected to adhere to ethical practices such as ensuring user privacy, securing systems, and writing clean, maintainable code.

For more details, refer to the ACM Code of Ethics.

Ethical Considerations for Data Scientists

Data scientists must respect user consent, avoid bias, and ensure transparency when working with data.

More information can be found at the Data Society website.
